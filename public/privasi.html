Kebijakan Privasi

Tukang Ngukur — Aplikasi Survey Profesional
Berlaku efektif: 10 Juli 2026 · Terakhir diperbarui: 10 Juli 2026

Kebijakan ini menjelaskan bagaimana aplikasi Tukang Ngukur ("Aplikasi", "kami") mengumpulkan, menggunakan, menyimpan, dan melindungi data Anda. Dengan menggunakan Aplikasi, Anda menyetujui praktik yang diuraikan di sini.

1. Data yang Kami Kumpulkan

a. Data akun (autentikasi)

Saat Anda masuk, kami memproses alamat email dan nama Anda — diperoleh melalui Google Sign-In atau verifikasi kode (OTP) email. Data ini digunakan untuk mengenali akun Anda, mengamankan sesi, dan mencegah penyalahgunaan (batas kuota pengguna).

b. Data survei (dibuat oleh Anda)

Koordinat titik, foto, catatan, dan atribut pengukuran yang Anda buat disimpan secara lokal di perangkat Anda (penyimpanan aplikasi/IndexedDB). Data ini hanya dikirim ke server apabila Anda secara sengaja mengatur "Server & Sinkronisasi" untuk integrasi dengan perangkat lunak kantor (mis. QGIS). Tanpa pengaturan itu, data survei tidak pernah meninggalkan perangkat.

c. Lokasi (GPS)

Aplikasi mengakses lokasi perangkat untuk menampilkan posisi Anda di peta dan melakukan pengukuran lapangan. Pemrosesan lokasi dilakukan di perangkat; lokasi tidak dikirim ke server kami kecuali menjadi bagian dari data survei yang Anda sinkronkan.

d. Cuaca

Untuk menampilkan ramalan cuaca lapangan, koordinat perkiraan Anda dikirim ke layanan cuaca pihak ketiga (Open-Meteo).

2. Bagaimana Data Digunakan

  • Memberikan akses masuk dan menjaga keamanan akun Anda.
  • Menyimpan dan menampilkan data pengukuran Anda.
  • Menyinkronkan data ke server pilihan Anda (opsional).
  • Mencegah spam/penyalahgunaan (pembatasan laju pengiriman kode).

Kami tidak menjual data pribadi Anda dan tidak menggunakannya untuk iklan.

4. Penyimpanan & Retensi

  • Di server kami: hanya email Anda (untuk kuota & keamanan) dan sesi login sementara. Kami tidak menyimpan foto atau data survei di server kami kecuali Anda menyinkronkannya ke server yang Anda atur sendiri.
  • Di perangkat Anda: data survei dan sesi login disimpan lokal hingga Anda menghapusnya atau menghapus akun.

5. Hak Anda & Penghapusan Data

Anda dapat menghapus akun dan seluruh data kapan saja:

  • Melalui aplikasi: buka Profil → Hapus Akun. Ini menghapus akun Anda dari server kami dan seluruh data lokal di perangkat.
  • Melalui web: lihat halaman Hapus Akun & Data.
  • Melalui email: kirim permintaan ke egi.geomatika@gmail.com.
Menghapus akun bersifat permanen: email Anda dihapus dari daftar pengguna kami, dan data survei di perangkat ini akan dihapus.

6. Keamanan

Koneksi menggunakan HTTPS terenkripsi. Token sesi ditandatangani secara kriptografis. Kode verifikasi email hanya berlaku 10 menit dan dibatasi jumlah percobaannya.

7. Anak-anak

Aplikasi ditujukan untuk penggunaan profesional (survei/pengukuran) dan tidak ditujukan bagi anak di bawah 13 tahun.

8. Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan penting akan ditandai dengan pembaruan tanggal di atas.
