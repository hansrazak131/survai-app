Drilling MachineField checker

Produktivitas Drilling Machine

Input dibuat ringkas untuk pengawas lapangan, tetapi output tetap detail: produktivitas, drillmeter, BCM coverage, KPI maintenance, quality drilling, dan AI Advisor.
Panduan inputPilih mode Open Pit atau Underground, pilih unit, lalu isi desain dan waktu aktual. Gunakan Load Example untuk melihat contoh target tercapai/belum tercapai, maintenance warning, dan quality issue.

1. Identitas & Unit

Data ini membantu laporan harian mudah ditelusuri berdasarkan unit, operator, area, material, dan kondisi lapangan.
Database unit umum Indonesia: Top Hammer, DTH, Rotary, Jumbo, Longhole.

2. Desain Drilling

Untuk open pit, produktivitas dihitung dari burden × spacing × bench height, cycle time per hole, penetration rate, dan efficiency.
Untuk underground, output utama memakai meter/shift, round/day, advance/day, dan quality drilling. Input burden/spacing dapat diabaikan.

3. Cycle Time & Kondisi Batuan

Penetration rate aktual dikoreksi oleh kondisi batuan dan kondisi bit. Cycle time per hole dihitung dari waktu bor + setup + pindah lubang + rod + flushing + delay.

4. Maintenance, Delay & Quality

Bagian ini dipakai untuk mencari penyebab target tidak tercapai: breakdown, delay, bit wear, hole collapse, deviation, dan kualitas lubang.

Produktivitas wheel loader

Hitung produktivitas Wheel Loader per jam dan estimasi harian berdasarkan bucket capacity, fill factor, cycle time, swell factor, densitas, job efficiency, EWH, dan target fleet.
Panduan inputPilih unit dan material, lalu parameter material otomatis mengikuti database. Nilai tetap dapat diedit sesuai aktual lapangan, hasil time study, kondisi stockpile/front, dan target produksi.
Database Indonesia: Komatsu WA150–WA900, Hitachi ZW220, CAT 966 GC, dan Custom.
Material mengisi FF, SF, density, dan efficiency otomatis.
Kapasitas bucket loader sesuai spesifikasi atau hasil kalibrasi lapangan.
Gunakan CT total dari time study atau aktifkan Advanced CT untuk menjumlahkan komponen.
Jam kerja efektif per hari/shift.
Target sebagai pembanding status target dan kebutuhan unit.
