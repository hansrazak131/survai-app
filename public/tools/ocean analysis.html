📊 Dashboard Summary

Ringkasan seluruh data Ocean & Direction Analysis yang sudah dimuat: Current Rose, WRPlot 8/16/32, dan ODV/CTD Profile.

🧭 Modul Tersedia

Pilih modul untuk import data dan menjalankan analisis.

📥 Import Data TXT / CSV / Paste Excel

Tempel data dari Excel, upload file TXT/CSV/TSV, atau pakai data sample bawaan. Delimiter dan kolom dideteksi otomatis.

👀 Preview & Pemetaan Kolom

🌊 Current Rose

Analisis arus: komponen N/E, residual/tidal current, dan arah dominan. Input minimal: Time, Direction, Speed.

🧭 WRPlot 8/16/32 Mata Angin

Klasifikasi arah & kelas kecepatan, frequency table, dan rose chart. Input minimal: Direction, Speed.

Masukkan batas atas tiap kelas (m/s, knot, atau cm/s sesuai data Anda). Kelas pertama otomatis "Calm" (~0).

🌡️ ODV / CTD Profile Plot

Profil oseanografi: depth vs temperature/salinity/conductivity/density/sound speed, QC, dan statistik per stasiun.

🛰️ Skyplot / Vector Plot

Visualisasi polar arah & magnitudo dari data Current Rose atau WRPlot yang sudah dihitung.
