Radial Bearing & H-Distance by Coordinates
Compute the Bearing & Distance from any station point to other points. This program removes the need to keep re-entering the first set of co-ordinates.
PROGRAM CODE
Deg:"STN E"?A:"STN N"?B:"SCALE FACTOR"?Z Lbl 0:"POINT E"?C:"POINT N"?D "BEARING=":Arg((B-D)+(A-C)i)+180►DMS "H-DIST=":Abs((B-D)+(A-C)i)÷Z Goto 0
INPUT DATA
⚠ Please fill all fields.
DIAGRAM
OUTPUT
BEARING — ° ' "
H-DISTANCE —
EXAMPLE: STN E=4.00 STN N=5.00
POINT E=456.547 POINT N=556.547
SCALE FACTOR=0.999621

Triangles
Compute Sides, Angles & Area of a triangle. Three options: [1] SSS — three sides known, [2] ASS — angle A + sides B, C, [3] SAA — side A + angles B, C.
