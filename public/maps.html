GIS Survey Controller

Kontrol citra dan layer langsung di window peta. Kalibrasi dan display memakai tombol khusus di sisi peta.

Default: Google Citra untuk tampilan citra/satelit. ESRI Satellite, OSM, dan Topo tetap tersedia sebagai pilihan.
Layer manager khusus untuk pengaturan layer: show/hide, lock, opacity PDF, rename, zoom, dan hapus/sembunyikan. Slider opacity dipindahkan ke sini agar area peta bersih dan premium.
Kalibrasi langsung 2–4 titik kontrol: tekan Tambah Titik Kontrol, tap titik pada PDF overlay, lalu tap posisi yang benar pada citra satelit.
Minimal 1 titik = geser overlay. 2–4 titik = geser + skala kasar berdasarkan pasangan kontrol. Untuk rotasi/skew presisi seperti Avenza asli, tahap berikut perlu affine/warping raster.

Layers & Basemap

Pilihan citra satelit dipindahkan ke menu layer agar lebih rapi dan hidden.

Catatan: WorldView/Maxar asli biasanya perlu API key / URL tile khusus. Kolom placeholder sudah disiapkan di script: WORLDVIEW_TILE_URL.

Kalibrasi Bounds PDF

Atur batas koordinat PDF agar overlay pas dengan satelit.

Cara cek: turunkan opacity PDF ±50%, pilih ESRI Satellite, lalu sesuaikan batas sampai sungai/jalan PDF menempel dengan citra satelit.
